Multi-Link

Multi-Link bietet einen Link um mehrere Twitch Kanäle in einem UI zu teilen. Toll für Rennen, oder andere Wettbewerbe. Gib eine, durch Leerzeichen getrennte, Liste von Kanälen an. Lege einen Zeitintervall fest, welcher den Chat in regelmäßigen Abständen wissen lässt, dass Multi-Link aktiviert ist.

Streamer bekannt machen

Einen Streamer bekannt machen! Zeigt im Chat dessen Twitch URL und das Spiel, welches gerade gespielt wird, oder zuletzt gespielt wurde.
