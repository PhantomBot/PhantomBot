This OAuth setup page is deprecated.

It will remain here while it is still required to initially setup the bot, but may be removed in the future.

This page may be missing some OAuth scopes required for newer features of PhantomBot added in v3.5.0 or later.

The recommended course of action is to use this page in the initial setup of the bot and to get around any invalid token errors, then to go to the bots web server (http://localhost:25000) and setup automatically refreshing OAuth tokens

NOTE: Tokens obtained through this page will expire after approximately 3 months. We strongly recommend setting up automatically refreshing tokens from the OAuth page that is built-in to your bots webserver.

You will need to setup the OAuth from this page first, then go to the bots built-in web server, usually located at http://localhost:25000 or similar.
